# Curriculum_backend
Backend para la creación de un curriculum

# Modulos

Incluyen los siguientes módulos:

1. Perfil: Información personal, foto, descripción, etc.
2. Experiencia: Experiencia laboral, proyectos, etc.
3. Educación: Información académica, títulos obtenidos, etc.
4. Habilidades: Habilidades técnicas y blandas.
